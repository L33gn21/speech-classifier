05 확정된 원인(버그 2개) & 적용한 수정

버그 ① pos_conv 랜덤 초기화

transformers 4.44.2+torch 2.4.1이 신형 parametrizations.weight_norm(original0/1)을 쓰는데 wav2vec2-base 체크포인트는 구형 weight_g/weight_v → 키 불일치로 pos_conv 미로드(랜덤). (HF #27605)

수정: _legacy_weight_norm()로 구형 weight_norm 강제. 결과: 경고는 사라졌으나 과적합은 여전히 실패 → 이것만으론 부족.

버그 ② gradient checkpointing (진짜 병목)

백본 config의 gradient_checkpointing=True가 자동 활성화 + reentrant 모드. 하위 동결 레이어의 체크포인트 입력이 grad를 요구하지 않아 gradient가 조용히 끊김("Gradients will be None") → 언프리즈한 상위 레이어가 학습되지 않음.

수정: 백본 checkpointing 강제 OFF + 학습 전 across-clip 피처 분산 계측 추가. 재빌드 후 tiny-overfit 재검증 중 — loss가 0으로 붕괴하면 성공.
